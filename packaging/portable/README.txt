Aether Portable 使用说明
==========================

1. 解压后，请直接运行同目录下的 Aether.exe。
2. 首次使用时，程序会在 %LOCALAPPDATA%\Aether\config.json 自动生成正式配置文件。
3. 包内的 config.example.json 只是示例模板，不会被程序直接读取。
4. 如果你想预先准备配置，请手动复制 config.example.json 到 %LOCALAPPDATA%\Aether\config.json 后再启动程序。
5. Aether.exe、aether-core.exe、wintun.dll 必须保持在同一个目录，不能单独移动或删除。
6. 点击 GUI 中的“启动代理”后，系统可能弹出权限提升确认，这是正常行为。

建议：
- 用 GUI 里的“打开配置文件”快速定位正式配置。
- 若升级便携版，直接覆盖程序目录即可；用户配置仍保留在 %LOCALAPPDATA%\Aether。
